# 💕 今天吃什么 - 女朋友专属美食决策平台

一个温馨有趣的 Streamlit 应用，帮助你的女朋友解决每天的终极难题：今天吃什么？

## ✨ 功能特色

### 🎲 随机推荐
- **智能推荐**：基于历史偏好的智能算法
- **转盘抽选**：酷炫的动画效果
- **完全随机**：让命运来决定

### 🎯 分类选择
- 中餐（川菜、粤菜、湘菜、家常菜）
- 西餐（意式、美式、法式）
- 日韩料理
- 快餐小吃
- 甜品饮品

### 🎮 互动游戏
- **抽签系统**：抽美食运势签
- **掷骰子**：随机决定美食类型
- **爱心猜猜猜**：趣味猜谜游戏

### 💖 个性化功能
- 收藏喜欢的美食
- 设置不想吃的黑名单
- 历史记录追踪
- 数据统计分析

## 🚀 本地运行

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行应用：
```bash
streamlit run app.py
```

3. 在浏览器中打开 `http://localhost:8501`

## ☁️ 部署到 Streamlit Cloud

### 步骤 1：准备 GitHub 仓库

1. 在 GitHub 上创建一个新仓库
2. 将以下文件上传到仓库：
   - `app.py`
   - `requirements.txt`
   - `README.md`

### 步骤 2：部署到 Streamlit Cloud

1. 访问 [share.streamlit.io](https://share.streamlit.io/)
2. 使用 GitHub 账号登录
3. 点击 "New app"
4. 选择你的仓库
5. 设置：
   - **Repository**: 你的仓库名
   - **Branch**: main
   - **Main file path**: app.py
6. 点击 "Deploy!"

### 步骤 3：分享给女朋友

部署完成后，你会获得一个公开的 URL（如：`https://your-app.streamlit.app`），把这个链接分享给你的女朋友就可以了！

## 🎨 自定义

你可以轻松修改 `app.py` 中的内容：

- **添加更多美食**：编辑 `FOOD_DATABASE` 字典
- **修改颜色主题**：调整 CSS 样式部分
- **添加新功能**：在现有标签页或新建标签页中添加

## 🛠️ 技术栈

- **Streamlit**: Web 应用框架
- **Pandas**: 数据处理
- **Python**: 后端逻辑

## 📝 注意事项

- 部署到 Streamlit Cloud 是完全免费的
- 应用数据存储在浏览器 session 中，刷新页面会重置
- 如需持久化数据，可以考虑集成数据库（如 SQLite）
